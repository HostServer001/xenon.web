Desktop helping program

Welcome to Xenon's web page, a program designed in python to make your desktop experience smoother and more efficient.
Xenon can be a valuable tool for anyone looking to improve their desktop computer experience with a CLI program.


Features

-Whatsapp automation
-Local python server hosting
-File management commands
-Easily accessible calculator
-Window handling commands
-Utility commands
-Wikipedia in CLI
-Wifi controls

Installation

Here's How You Can Install Xenon

~Download the XenonSetup ZIP file by clicking on the button at the end of the section
~Download python from Mircosoft Store
~Unzip the XenonSetup file in Downloads folder itself
~Start PowerShell
~Type:- cd Downloads
~Type:- python XenonSetup
~Check your Desktop
~Start Xenon.bat by double clicking on it
~Wait until it installs everything
~Reopen thw same file -> enter credentials -> restart Xenon
Done!
